# CaseFlow

CaseFlow is an internal case and task management system designed to simulate real operational software used inside organisations.

## Features

- Case creation and tracking
- Priority and status management
- Case notes system
- Dashboard overview
- Clean internal tool interface

## Tech Stack

Frontend:
- React
- Vite

Backend:
- Node.js
- Express

Database:
- PostgreSQL

## Architecture

React → Express API → PostgreSQL

## Purpose

This project is part of a software development portfolio focused on building practical internal tools rather than generic demo apps.

## Future Improvements

- Authentication and roles
- Search and filtering
- Activity timeline
- Analytics dashboard
